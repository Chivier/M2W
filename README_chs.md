# M2W Project

## 什么是M2W

偷懒是第一生产力（逃）

M2W的项目目的在于讲视频文件和纯文本进行互相转换。提供便捷、快速的工具链，实现对视频文件的“变相压缩”。当下解决的问题相对局限，我们希望解决讲座视频的极致压缩。例如在听讲座的时候，我们更加关心讲台上的教授在说什么，他的PPT是什么，对于他的长相、动作我们并没有那么在乎。所以我们希望将这种特殊类型视频文件进行极致的压缩。
